プライバシーポリシー

個人情報保護方針

「クルカツ」（以下「本サービス」といいます。）の運営者である組嶽和樹（以下「運営者」といいます。）は、本サービスにおける利用者の個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます。）を定めます。

本サービスは、原則として18歳以上の大学生・大学院生を対象としており、サークル活動の管理・コミュニケーションを支援することを目的としています。

制定年月日: 2025年8月16日

最終改正年月日: 2025年9月5日

クルカツ運営者: 組嶽和樹

第1条（基本方針）

運営者は、利用者の個人情報の重要性を認識し、個人情報保護法その他の関連法令を遵守し、適切な個人情報の取得、利用、管理を行います。

第2条（収集する個人情報）

運営者は、本サービスの提供に必要な範囲内で、以下の個人情報を取得します。

1. アカウント登録情報

  • 氏名、メールアドレス、パスワード
  • 大学名、学年、性別、生年月日

2. プロフィール情報

  • プロフィール画像
  • 氏名、大学名、学年、性別、生年月日

3. サークル関連情報

  • サークル名、活動内容
  • 活動頻度、活動曜日
  • サークル紹介文
  • 代表者・メンバーに関する情報
  • サークル画像

4. コミュニケーション情報

  • メッセージ内容、画像等のユーザー生成コンテンツ
  • イベントの説明文
  • 報告・ブロック情報

5. 利用状況情報

  • Firebase認証情報（ユーザーID、メールアドレス）
  • Firestoreデータベースへのアクセス記録
  • プッシュ通知の受信・開封状況
  • アプリの利用頻度・機能利用状況

6. 通知関連情報

  • プッシュ通知トークン（Expo Push Token）
  • 通知設定（入会申請通知、サークル連絡通知）
  • デバイス情報（iOS/Android識別）

第3条（個人情報の利用目的）

運営者は、取得した個人情報を以下の目的のために利用します。

  1. 本サービスの提供・運営
    • アカウント管理、認証・認可
    • サークル検索・管理・マッチング機能の提供
    • メッセージ・コミュニケーション機能の提供
  2. セキュリティ・不正利用防止
    • 本人確認、なりすまし防止
    • 不正アクセス・不正利用の検知・防止
    • アカウントの安全性確保
  3. ユーザー生成コンテンツの管理
    • 不適切コンテンツの検出・除去
    • 24時間以内の対応実施
    • 報告されたコンテンツの確認・評価
  4. サポート・コミュニケーション
    • お問い合わせ対応、サポート提供
    • 重要なお知らせの配信
    • サービス改善のためのアンケート実施
  5. サービス改善・開発
    • 利用状況の分析・統計
    • 新機能・キャンペーンの案内
    • サービス品質の向上
  6. 法令・規約対応
    • 法令に基づく対応
    • 利用規約違反への対応
    • 権利侵害への対応

第4条（個人情報の適切な取り扱い）

1. 運営者は、個人情報の取り扱いについて、以下の措置を講じます。

  • 個人情報への不正アクセス、漏洩、滅失、毀損の防止
  • 個人情報の正確性・最新性の確保
  • 不要となった個人情報の適切な削除

2. 運営者は、本サービス運営のため、以下の外部サービスを利用しています。

  • Firebase（Google LLC）: データベース・認証・分析サービス
  • Expo Push Notification Service: プッシュ通知配信サービス
  • Google Analytics for Firebase: 利用状況分析サービス
  • その他、本サービス運営に必要な外部サービス

3. これらの外部サービスは、各社のプライバシーポリシーに基づいて個人情報を適切に取り扱います。

第5条（個人情報の第三者提供）

1. 運営者は、以下の場合を除き、利用者の同意なく個人情報を第三者に提供しません。

  • 法令に基づく場合
  • 利用者の明示的な同意がある場合
  • 人命・財産保護のために必要で、本人の同意取得が困難な場合
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要で、本人の同意取得が困難な場合

2. 運営者は、本サービスの機能提供のため、以下の範囲で個人情報を他の利用者に表示することがあります。

  • サークルメンバー間でのプロフィール情報の共有（氏名、大学名、学年、プロフィール画像）
  • サークル検索結果での基本情報の表示
  • メッセージ送信者名の表示

第6条（ユーザー生成コンテンツの取り扱い）

1. 利用者が投稿したメッセージ、画像等のユーザー生成コンテンツには、個人情報が含まれる場合があります。

2. 運営者は、不適切コンテンツの報告・対応のため、以下の情報を利用することがあります。

  • 報告されたコンテンツの内容・投稿者情報
  • 報告者の情報（匿名化の上）
  • ブロック関係の情報

3. 運営者は、24時間以内の対応を確実に行うため、報告されたコンテンツに関する情報を適切に管理します。

第7条（利用者の権利）

利用者は、運営者が保有する自身の個人情報について、以下の権利を有します。

  1. 開示請求: 保有する個人情報の内容確認
  2. 訂正・追加・削除請求: 不正確・不完全な情報の修正
  3. 利用停止・消去請求: 個人情報の利用停止・削除
  4. 第三者提供停止請求: 第三者への提供停止

これらの請求は、第11条に定める窓口までお問い合わせください。運営者は、法令に基づき適切に対応いたします。

第8条（ローカルストレージ・類似技術の利用）

1. 本サービスでは、利用者の利便性向上のため、以下の技術を利用しています。

  • AsyncStorage: オンボーディング表示制御、Firebase認証状態の保持
  • Firebase認証の永続化機能
  • 通知設定の保存
  • アプリ内の設定情報の保存

2. これらの技術により、アプリの設定や認証状態が端末内に保存されます。

3. 利用者は、アプリの再インストールにより、保存されたデータを削除することができます。

4. 通知トークン等の重要な情報は、Firestoreに安全に保存され、端末内には最小限の情報のみが保存されます。

第9条（未成年者の個人情報）

1. 本サービスは、原則として18歳以上の大学生・大学院生を対象としています。

2. 18歳未満の方が利用する場合には、保護者の同意を得てから利用してください。

3. 運営者は、未成年者の個人情報について、特に慎重に取り扱い、適切な保護措置を講じます。

第10条（プライバシーポリシーの変更）

1. 運営者は、必要に応じて本ポリシーを変更することがあります。

2. 変更後は、本サービス上に掲示した時点から効力を生じるものとします。

3. 重要な変更がある場合は、利用者に事前に通知いたします。

4. 外部サービスの追加・変更等、個人情報の取り扱いに影響する変更がある場合は、必ず事前に利用者に通知いたします。
